# Tocco User Testing Protocol

## Participants

- 1-3 users familiar with basic 3D navigation.
- Test on one iPhone and one iPad.

## Task Script

1. Place AR clay mesh in front of user.
2. Create a shape with at least:
   - one raised region (`pull`)
   - one carved region (`push`)
   - one smoothed blend (`smooth`)
3. Use `undo` and `redo` at least once.
4. Switch to `Navigate` mode and rotate/scale the model.
5. Save session, reload session, and export OBJ.

## Metrics To Record

- Task completion time (minutes).
- Number of accidental edits (self-reported + observed).
- FPS stability (from overlay).
- Perceived control score (1-5 Likert).
- Perceived UI clarity score (1-5 Likert).

## Debrief Questions

- Which tool felt most intuitive and why?
- What interaction felt least reliable?
- Did touch-only controls feel sufficient?
- What would make this feel closer to physical clay?
